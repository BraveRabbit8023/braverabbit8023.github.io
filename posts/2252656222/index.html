这是一篇从零开始关于科学上网/梯子/魔法的帖子，包括一些简单的科普（但不严谨）和一些常用名词的解释。内容很长但是足够有用，几乎可以解决日常浏览大千世界遇到的阻碍，如果你有耐心看完的话。

简介

因为政策限制，通常情况下我们运营商提供的网络服务是没办法直接访问那些被 BAN 掉的网址链接。我们每发起一次访问都会被审查，当发现目标 IP 处在“黑名单”时这次访问就会被拒绝从而失败，因此我们需要利用所谓“科学上网”来绕过这些限制。

科学上网简介

科学上网的主要原理是通过加密、代理和伪装等技术绕过网络审查,访问被封锁的网站和信息。具体来说:

  1. 加密：将网络数据流进行加密,使审查系统无法判断你在浏览什么内容。常用的加密协议有 SSL / TLS 等。
  2. 代理：通过第三方代理服务器与目标网站建立连接,将流量先发送到代理服务器,再由代理服务器转发到目的地,从而绕过审查设备访问互联网。常见的代理协议有 SOCKS、HTTP 代理等。
  3. 伪装：将真实的网络流量伪装成其他合法的流量,混淆在正常数据中传输,从而逃避审查检测。
  4. 虚拟通道：科学上网工具在防火墙之间架起一条虚拟通道,网络流量通过这条通道代理转发,绕过审查访问被封锁的网站。

常见的科学上网工具如 Shadowsocks（小火箭）、VPN 等,都是基于上述原理,通过加密、代理服务器等手段帮助用户访问受限制的网络资源。

工具形式简介

无论是哪种形式，都会有一些第三方来提供这些服务。但这些服务对于普通人日常使用来说都是无法直接作用在设备上（其实能但是很复杂），需要一个工具来使这些代理节点或者加密手段生效（就好像你光安装一个 Python 环境，但是没有编辑器或编译器你也没法写出代码并运行）。

集成式梯子——傻瓜式 App

一些商家会进行一体式开发，提供服务的同时会直接集成到一个 App 上，比如“旋风加速器”、“快连”、“夏时”这类，下载安装后选择好节点区域直接点击“立即连接”这种按钮就可以一键科学。类似瓶装水，但是农夫山泉的瓶子只能装农夫山泉的水。

  • 优点：快捷省力方便、如果是正规公司的服务，或有质量和一定的售后保障。
  • 缺点：不一定多端通用，可能只开发了 IOS 或者安卓，没办法多设备共享；可操作性低；价格相对高，因为要有开发和维护的成本。

代理节点——机场 + 代理工具

还有很大一部分是提供代理节点的集合，我们称之为“机场”。但是不提供 App 工具，你需要自己去寻找能够使这些服务或产品生效的工具。类似自来水，你只要有瓶子装就行，不管什么瓶子。

  • 优点：服务本身不受限于 App（因为根本没有嘿嘿），只要在任何设备上找到合适的代理工具进行部署导入就可以使用，自由度高，可操作性强（如果不限制设备且流量够多你可以整八百个人一起用，我就是这么干的，“一处水源供全球”）；此外同质量线路下，机场要比集成式开发相对便宜。
  • 缺点：你要自己去找提供服务的网站，但是这种灰产本身就不在墙内，就会出现“想买机场，要先翻墙，要想翻墙，得买机场”的死循环；很多便宜机场是个人散户开的，或者是从废弃机场搬来的，经常有跑路的可能性（一分钱一分货到哪都不假）；没有那么傻瓜，步骤相对繁琐一些。

至于选择哪一种，那就是朋友们的个人情况了，但从优缺点的详略程度上大家也能看出兔兔目前在用的是哪一方案吧 [Doge]~

方案教程

教程主打一个详略得当，我会把各种方法都尽量说一说，但最后会着重介绍兔兔在用的方案。总体流程是由简到繁，从 Free 到花钱，维度上会从主要的三个系统来介绍：IOS、安卓以及 Win（Mac 差不多）。

傻瓜免费方案

这个方案就给只给大家介绍手机端了，毕竟我在电脑上找免费梯子的日子已经是好多年前了，那时候手机还不是长时间上网设备。现在嘛……我知道的那些免费 App 早就寄了，现在也不用了所以就只介绍一下手机端吧。

安卓

安卓机器我也有年头没用了，但奈何经常搞这些，大家会互相推荐。目前我知道的最为广泛使用的是“旋风加速器”，应该是主打一个每天两小时免费使用，简单刷刷刷应该是够了；此外当年还有一些比较经典的现在或许还能使用：“老王加速器”、“银河”等

IOS（苹果）

因为苹果对自家商店的管制，App Store 是有区域划分的，很多能用的梯子根本不可能上架国区的。因此在下载那些可用的 App 之前我们最好是拥有一个外区账号。外区账号可以自己注册，也可以购买成品号（但我觉得没有必要）。

😎 这里只介绍如何翻墙，如果需要注册外区账号的可以自己去 XHS 或者 某乎找教程。懒得自己弄的可以滴滴我（点击主页微信图标扫码滴滴）手动注册外区号。

IOS 外区能下载到的梯子有很多，免费的也不少。但是有一部分免费的梯子时好时坏，还有的用着用着就收费了（特此点名“小牛加速器”），目前我手机里留着的就只剩下“夏时 ”和“银河”这两个加速器了。延迟大且经常断连，但奈何是免费的，将就用吧！

代理节点 + 代理工具方案

代理我们前文提到过，就是通过第三方可访问的服务器建立连接，然后你直接访问这个服务器就会把你的流量转发到目标网站上。那这一个个可供连接并转发的“跳板”就称之为节点。

一个节点里可能包含它所在的服务器、加密协议、访问许可等一系列代码和规则信息，你需要一个能帮助你理解并部署这些节点的工具，使其生效，这就是我们要使用的代理工具。

关于代理节点

由某商家运营的代理节点集合我们一般称为“机场”，运行的商家个人或团队我们称为“机场主”。有很多无主的免费代理节点，由热心人士打包成一个集合，这种我们也可称为“机场”，但我一般就叫它“免费节点集合”，因为“机场”能提供服务，需要有人运营，这种散落的节点很明显不在此列。

前文不是提到过很多机场的网址都是不可直接访问的嘛，这个时候我上面说的那些免费的集成式 App 就派上用场了。他们的作用就是时刻准备着，当现役方案爆炸的时候保障你还能有一丝生机访问外面的世界。

机场的主要使用方式就是订阅制，通过“Subscribe”来导入到你的代理工具中，99% 的机场网站都会提供“一键导入”功能。不同的套餐费用也代表了节点的质量或者其他服务的限制，比如像流量多少、是否限速、是否解锁流媒体、是否限制设备数、可用节点数量等。

下面是兔兔目前在用的机场，推荐给大家：

  • FlyingBird：平时一直在用这个，速度和延迟都还可以。
  • Hutao Cloud：备用选项，怕哪天飞鸟不好使了没得用。
  • STC-Serve：质量比前两个要高，当然也要贵，只是在一些要求节点质量的时候使用。

❗ 注意：机场有时会出现跑路现象，兔兔之前遭遇过几次，但是作为灰产这也是没办法的事情。这也是为什么我不推荐太便宜机场的原因——怕跑路；也不推荐太贵的原因——贵的一旦跑路更亏！

关于代理工具

代理工具就是能够使代理节点在各设备上生效的 App，每个平台主流使用的就只有那么几个，如果有的同学是在一些商家直接买的节点，他们一般也就是使用这几种代理工具。

安卓 & PC

这两个系统放在一起讲的原因是：我最最最最常用的 Clash 同时开发了适用多个系统的版本。虽然 Clash 开发者因为某些原因停止了对该项目的维护，但是只是不再迭代以及 Debug，软件本身依旧能正常使用。

  • Clash for Windows
  • Clash for Android
  • Clash for Mac

此外这三端还有其他代理工具如：V2ray 系列、Bypass 等。我推荐 Clash 的主要原因是我喜欢一个系列的东西（但是 IOS 我实在没办法噜）；此外就是 Clash 的 PC 端相较于 V2ray 的 UI 更美观一些。

其中对于 Android 而言 V2rayNG 可能是更好的选择，Sing Box 也是可以的，因为 Clash 的代码可能不太精简，据说还有 DNS 泄露的风险，但是这些对于普通使用者来说不是需要考虑的东西。这几个工具应该是在 Google 应用商城上架了，所以就不提供安装包文件了，ApkPure 也可以下载。

IOS（苹果）

IOS 也有很多代理工具，比如Quantumult X、Surge、Loon、Shadowrocket 等。软件本身各有差异，功能也不尽相同，但是绝对统一的有两点：

  1. 国区无法下载，需要外区账号。
  2. 没有一款是免费的都要花钱，外区账号又不支持国内付款方式。
代理工具 价格
Surge 首年 $50 + 后续 $15 / Year
Quantumult X $8
Stash $3.99
Shadowrocket & Loon $3

其中使用最广泛的可能是 Shadowrocket，俗称“小火箭”，因为支持最多最全面的协议，价格最低，成为性价比之王~
因为即使注册了外区账号，付款也是一大难题（有的同学可能没有），那么要止步于此吗？当然不，曲线救国的方法是想办法（免费 or 付费）获得一个已经购买了 Shadowrocket 的外区账号登陆自己的 App Store，下载完成之后再退出。

作为坚持白嫖好多年的勇敢兔兔肯定不会去买帐号了，毕竟代理工具又不需要经常更新，下载一次能用好久。所以这里给大家放出一个小火箭的账号共享网站，需要的同学可以微信滴滴我！

写在最后

一般来说比较灵活的使用方法是：

  1. 质量较好、跑路可能性小的机场作为主力机场使用。可购买包年套餐长期使用。
  2. 质量最好、创立多年的优质机场作为备用机场。可按量付费或按月付费备用。
  3. 至少保证一个设备中有可用的免费梯子，作为救急方案。

另外在一些审查严格的时期绝大多数的机场或梯子都处在半瘫痪状态，不可强求。

注意 ChatGPT 这种对于 IP 审查严格并有封禁的网站，访问时要尽量保证节点一致性，不要随意频繁切换；同时注意机场是否提供解锁保证。

“流媒体解锁”是解除部分视频提供商的地区限制，有些廉价机场不提供这一服务导致有些视频无法观看，注意辨别。

最后希望大家都能“开眼看世界”，Bye了个Bye！
