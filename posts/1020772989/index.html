“什么！！！我的 C 盘又红了？”

前言

某一天，当你打开“我的电脑”，发现一条醒目的红色跃然屏上，那么刺眼、痛心！

随着目光继续下移，你的表情逐渐疑惑且凝重，陷入了深深的思考……

是的没错，剩下几个盘符甚至空的都没有什么蓝色，主打一个陪伴。我们“袓國菗潒哌埖朶”曾有言：

红色是毁灭❤️，蓝色是冷漠💙，白色是虚无🤍……

你甚至一时间都想不出为什么，这电脑后宫佳盘 1 个 T，偏宠 C 妃一人。
这时有的同学灵机一动，打开了“XX 电脑管家”-“清理垃圾”，一顿闪烁找到了好几个 G 的垃圾，全部清除后发现任尔风吹浪打， C 盘岿然不动。

如何拯救你的 C 盘（文末有懒人方法）？

这里不直接给大家介绍 C 盘的组成部分了，感兴趣的可以自行找科普。我只给大家列出最有可能使 C 盘增长的几个主要元凶：

放满文件的桌面

有些同学可能会疑惑：“桌面都不在“计算机 or 我的电脑”里，怎么会占 C 盘空间？”很不幸，桌面本质只是一个视窗，它的实际默认路径是 “C:\Users\你的用户名\Desktop”。仔细想想，你放在桌子上的东西就没有占用你房间的空间吗？

如果你的桌面只是一些快捷方式（带箭头的小图标），那就算摆满也占不了多少储存空间。但是有的同学会经常把下载的东西、自己的作业、各种素材甚至是软件安装包等直接扔在桌面（我接手过更离谱的还有直接把软件安装目录改成桌面的）。

不是说桌面不可以放文件，比如我图方便的时候也会在桌面直接新建文件。但是大家不能把所有的东西都一股脑堆在桌面上，比起“储藏室”桌面更适合成为“中转站”。

默认把软件全装在 C 盘

不是说软件不可以安装在 C 盘，毕竟固态硬盘不分区的话，那就只有一个盘。这里我们讨论的是常见情况都是要分区的，分区一般不会给 C 盘预留太大空间，如果 C 盘持续爆满，剩余空间不足 C 盘的 5%，才会影响运行速度。

但是毕竟很少有人能看见 C 盘“红温”而不难受（强迫症更得原地升天），所以我们在其他盘符空间充足的情况下尽量还是把软件安装在其他盘比较好。

安装时更改安装路径

一般软件安装程序主要有三种情况：

  1. 主动显示路径选择界面

  2. 折叠路径选择界面

    💢 现在很多国产软件都会增加一个“自定义选项”，其实一般就是一个路径选择 + 开机自启，有时候再来一个桌面图标。

  3. 不可修改
    这种情况并不多见，少数软件必需极好的系统支持或者运行速度，才会默认到 C 盘安装，人家不让改咱也就偶尔允许它安装那么一两次。但是可以更改的尽量更改。

已安装软件从 C 盘搬家

如果我们的很多软件已经安装在 C 盘，但是又不想删除再安装，有没有什么办法呢？当然有：

系统“设置”自带搬家

Win + i 打开“设置”
应用 - 安装的应用
点击选项（三个点）- 移动

❗ 注意：这种移动方式通常只适用于从 Microsoft Store 安装的软件，我们通过下载安装包进行安装的不在此列。

使用“搬家工具”

兔兔已知一些电脑管家里会提供这种工具，比如“360 卫士”或者“腾讯电脑管家”，网上也可以找到这些工具的纯净提取版。

文章末尾会附上下载链接。

微信 & QQ 聊天记录

腾讯旗下两款聊天软件都会默认把聊天记录放到 C 盘目录下，如果没有定期清理的习惯就会积累巨大量的内容。

其中包括文本聊天、传输的文件、图片、视频、语音甚至是你根本没有点击保存的表情包。

即使像兔兔这样定期清理，短期内也会迅速积攒很多文件（QQ 基本不用电脑版所以没有什么东西）。

综上所述，我们的应对方案有二：
首先是把微信 & QQ 的聊天记录储存路径更改到其他盘；
其次是定期清理。
下面只以微信为例进行示范。

储存路径更改

说明一下，我苯人是没改过路径的。
一个是因为我要经常下载各种软件或者从 Github 拉取一些东西，D、E 两盘满的比 C 盘快；
另一个是因为我本身有清理 C 盘的习惯，所以相当于聊天记录根本没在 C 盘存在过，也就没必要转移到其他盘。

一些杂项

电脑使用过程中也会有一些默认的基础行为会把文件存在 C 盘，比如浏览器下载的文件，系统自带的截图功能等。这些一般会默认分类存放在 “C:\Users\你的用户名” 文件夹下，而这几个分类通常也会默认固定在资源管理器的快速访问里，就像这样：

此外还有一点是大家容易忽略的，就是“回收站”，不是说把东西扔到“回收站”就不会占用空间了。

回收站类似一个临时文件夹，当你有东西在回收站时，它会生成一个隐藏文件夹来存放这些垃圾，以便你反悔复原。

写在后面

其实到现在这个时间节点，C 盘满与不满更多体现的是一个人的使用习惯而非真正意义上的技术。因为假设一开始就不分盘符，只有一个 C 盘，那也就无所谓了。

我迄今为止还是会“随意”地存放一些文件，是因为我有重要文件备份和定期清理的习惯，至于他们放在哪里对我的影响不是很大。而且作为个人用户，也没有什么可以称得上“不可丢失”的文件，有的只是搜寻和创作这些文件的时间成本与精力罢了。
