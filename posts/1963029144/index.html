百度给爷爬：PC 端百度网盘满速下载解决方案

世界上最抽象的事情莫过于百度网盘的限速。PC 端软件本身做的就又丑又不好用，下载又需要强制启动软件。一打开发现自己又不是会员，要么面对几 kb/s 的速度等一晚上，要么含泪充钱。那有没有什么办法可以绕过百度网盘的限速呢？

历史上，我们曾经拥有过 PanDownload、Kinh 等第三方客户端，但它们要么被百度官方所限制，要么已经无人维护而失效。好在无数好心人前仆后继开发的直链解析方法依旧迭代至今，可谓是“一个客户端倒下去，千万个解析站起来”。通过解析获得不限速直链，调用第三方下载器进行下载，这一方法在如今依旧有效，特此含泪感谢和百度抗争的开发者们！

现有解析链接的方法主要有两种：

  1. 网站解析：用户只需将百度网盘的下载链接复制到该网站上，即可转换成可供第三方下载器使用的链接。例如“就是加速”。
  2. 脚本解析：利用油猴脚本来更改百度网盘页面，从而内嵌脚本下载按钮，直接获取不限速的下载链接。

第三方下载器的选择有很多，目前最主流的是Motrix 和 IDM。

我的方案

我目前在用方案是：油猴脚本 + IDM（Internet Download Manager）。选择这一搭配方案的原因在于，它不仅提供了当下百度不限速解决方案，还考虑到未来这些工具的可持续使用。

比如油猴插件不仅能够提供解析百度网盘链接的功能，还能通过数以万计的脚本实现诸多功能：

  • 视频 VIP 解析
  • 超星安全教育自动播放答题（慎用！）
  • Bing 搜索结果去广告
  • ……

而 IDM 作为一款强大的下载器，可以替代迅雷等下载工具，提供多线程下载、自动按文件类型分类存放等实用功能。

安装教程

IDM 下载器下载教程

首先，访问 IDM 官网 并下载安装。完成后你可以享受30天免费试用。

关于如何绕过试用限制，我将尽快在后续的文章中提供教程。

油猴插件安装教程

❗注意：测试版（红色）和稳定版（黑色）均可，但部分其他脚本需要测试版才能运行。

Microsoft Edge 浏览器

  1. 打开 Edge，点击“扩展”。
  2. 点击“获取 Microsoft Edge 扩展”。
  3. 搜索“油猴”或“篡改猴”，下载安装。

Chrome 浏览器

  1. 进入“扩展”设置。
  2. 进入“Chrome 应用商店”（需要梯子访问）。
  3. 搜索“油猴”或“篡改猴”，下载安装。
无梯子替代方案
  1. 访问“Crx 搜搜”网站。
  2. 找到“篡改猴”并下载安装。
  3. 同时查看“如何安装扩展”并根据指引完成安装。

油猴脚本下载教程

  1. 在扩展工具栏中点击油猴图标。
  2. 选择“获取新脚本”（如果无法打开，可以直接点击帖子内链接跳转）。
  3. 访问 GreasyFork，搜索“神速”。
  4. 安装“神速 Down”脚本。

具体使用

用浏览器打开自己百度网盘，进入文件浏览页面。这时你会看到页面上多了“神速 Down”按钮。

选择想要下载的文件（最好一次选择一个），点击“神速 Down”按钮。初次使用需要配置 IDM 用户代理。

扫描二维码关注公众号，按照指引获得四位数字暗号，完成后点击上方“解析”按钮，在弹窗中输入刚刚获取的暗号，等待解析完成。

完成后点击“复制链接”，打开 IDM - 新建任务，会自动识别剪贴板内容并粘贴，开始任务即可。这时你会发现下载速度几乎可以达到满载。

一些唠叨

在前面我就提到过除了 IDM 还有一个 Motrix 下载器，这一点在“神速 Down”界面也有所印证。具体区别遵循“没必要，不了解”原则就不详谈了，有兴趣的可以自己探索研究。

另外，浏览器插件商城还有其他功能强大的扩展；油猴中我们又有海量脚本可以使用。浏览器插件和油猴脚本的合理使用可以把我们的浏览器变成生产力工具，大家也可以在这个基础上自行探索。
